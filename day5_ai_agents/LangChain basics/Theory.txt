---------- LangChain Basics ----------
LangChain is a Python and JavaScript framework used to build applications powered by Large 
Language Models (LLMs).

It helps developers create:
--> AI chatbots
--> AI agents
--> Workflow automation systems
--> Retrieval-Augmented Generation (RAG) systems
--> Autonomous reasoning systems
--> Multi-step AI pipelines

Instead of directly calling an LLM API every time, LangChain provides:
--> Chains
--> Agents
--> Memory
--> Prompt templates
--> Tools
--> Retrieval systems
--> Workflow orchestration

It helps connect:
--> LLMs
--> Tools
--> Memory
--> Chains
--> Databases

# LangChain Core Components
Component	               Purpose

# Limitations

Issue	                        Explanation
Complexity	                  Large workflows become difficult
Cost	                        API calls can be expensive
Latency	                     Multi-step reasoning takes time
Hallucinations	               LLMs may generate wrong info
Debugging	                  Agents can be hard to trace
