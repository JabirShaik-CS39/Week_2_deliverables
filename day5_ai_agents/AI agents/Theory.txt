----------- AI Agents -----------

# An AI Agent is:
An AI Agent is a software system that can:
--> Observe information from users, APIs, files, or environments
--> Reason about what to do
--> Plan actions
--> Use tools to complete tasks
--> Remember context or previous interactions
--> Act autonomously toward a goal

Unlike a normal chatbot that only replies with text, an AI agent can:
--> Search the web
--> Read PDFs
--> Send emails
--> Query databases
--> Call APIs
--> Execute code
--> Automate workflows
--> Make decisions dynamically

# AI Agent Architecture
User Goal
   ↓
LLM Brain
   ↓
Reasoning
   ↓
Tool Selection
   ↓
Action Execution
   ↓
Observation
   ↓
Repeat Until Goal Completed

# Key Characteristics of AI Agents
Feature	                Description
Reasoning	            Think step-by-step
Planning	            Break goals into tasks
Tool Usage	            Use APIs, DBs, search
Memory	                Remember context
Autonomy	            Act independently
Iteration	            Retry if failed
