---------- Document Ingestion ----------
Document Ingestion in RAG (Retrieval-Augmented Generation)

In a RAG pipeline, ingestion is the process of collecting documents, cleaning them, converting them into embeddings, and storing them in a vector database so that an LLM can retrieve relevant information later.

What is Ingestion in RAG?

Document ingestion is the first stage of a RAG system.

It transforms raw data like:

PDFs
DOCX files
Web pages
Text files
Database records

into searchable vector embeddings.

Complete RAG Ingestion Flow
